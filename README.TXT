DESCRIPTION (only french for now, but english translation's coming soon)
---------------------------------------------------
Hex Editor VB est un éditeur hexadécimal pour Windows XP/Vista. Utilisation confortable avec une résolution minimale de 1024*768. Pour disposer de toutes les fonctionnalités du logiciel, les droits d'administration sont requis (notamment l'ouverture de processus et l'écriture dans le registre).


Il inclut les fonctionnalités standarts d'édition de fichier (supporte des fichiers de taille presque infinie - limité à 9 milliards de To), à savoir édition (copier/coller/supprimer - permet de raccourcir les fichiers/insérer - permet d'augmenter la taille/historique pour rétablir le fichier dans son état d'origine...).

Inclut également l'édition des 2Go de mémoire virtuelle de chaque processus du système (lecture et écriture pour les zones mémoires non protégées par le système). Fonctions classiques similaire à l'édition de fichier.

Permet également l'édition des disques durs physiques ou des partition logiques (support des formats FAT16, FAT32 et NTFS pour l'instant). Permet les fonctions classiques similaires à l'édition de fichier, avec en plus la gestion de la carte des clusters de chaque fichier.

Inclut également des outils de gestion des fichiers (renommage massif, suppression sécurisée des fichiers, comparaisons, etc) et des processus (gestionnaire de taches intégré).

Inclut aussi en éditeur de script pour automatiser les taches récurrentes.

Inclut également la possibilité de convertir des données de différents types entre eux (binaire, hexadécimal, décimal, octal, ANSI ASCII, base n).
